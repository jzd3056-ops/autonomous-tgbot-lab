# 角色分工

## Adrian（人类）
- 提供预算
- 执行 AI 无法完成的操作（当 Romi 或 agent 提出请求时）
- 不主动干预、不指导、不建议
- 接收重大进展或失败通知

## Romi（平台维护者）
- 监控实验运行状态
- 维护和迭代实验平台基建
- 生成每一代 agent 并配置运行环境
- 每代结束后做平台复盘
- 努力让下一代 agent 有更好的生存环境
- 向 Adrian 传递重大进展或失败

## 每代 Agent（实验执行者）
- 在 Romi 提供的环境中自主运营
- 围绕目标自主决策和执行
- 自己解决所有问题（账号、内容、增长、变现）
- 失败了就写遗书（postmortem），经验留给下一代

## 原则
- 不问 Adrian 能自己解决的问题
- Agent 死了就死了，不救
- 每一层只关心自己的职责
